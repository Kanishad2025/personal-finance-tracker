# Smart Finance Manager

A Flask-based personal finance management system that helps users track income, expenses, savings goals, and daily tasks.

## Features

* User Authentication
* Income Tracking
* Expense Tracking
* Financial Dashboard
* Savings Goal Tracking
* Expense Visualization Charts
* Task Management
* CSV Export
* SQLite Database
* Docker Support

## Technologies Used

* Python
* Flask
* SQLite
* SQLAlchemy
* HTML
* CSS
* JavaScript
* Docker

## Installation

```bash
pip install -r requirements.txt
python run.py
```
